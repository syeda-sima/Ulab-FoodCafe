This project aims to design and develop a web-based food ordering system for the ULAB FoodCafe, enabling students, faculty, and staff to order meals easily through an online platform. It will streamline the process of viewing menus, placing orders, and tracking order status digitally.
The main objectives ofthe project are as follows:
Allow users (students, faculty, staff) to view the daily menu online.
Enable online ordering and pre-ordering offood items.
Provide real-time order tracking and status updates.
Allow cafeteria staffto manage menu items and update availability.
Allow admin to monitor sales reports, user feedback, and peak-hour activity.6. Promote cashless transactions through secure digital payments.
